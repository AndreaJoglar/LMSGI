Tipos de input:
1. <input type="text" name="xxxx">
Genera cajita estrecha para meter texto

2. <input type="submit" value="Enviar">
Genera un botón para el formulario para enviar info

3. <input type="reset" value="Comenzar de nuevo">
Genera un botón para borrar formulario

4.<input type="checkbox" value="xxxx">
Genera una cajita para marcar 

5. <input type="radio" value="xxxx">
Genera lo mismo pero en circular

Textarea: <textarea name="mensaje">
Genera una caja para escribir texto más largo
